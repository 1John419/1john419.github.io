Applications

All apps are open source, offline-first, and mobile-first. Once downloaded, an internet connection is not required.

All apps support Android, Linux, or Windows devices with the latest version of a Chromium-based browser.

All apps should also support the latest version of the Safari browser, but have not been extensively tested on Mac/iOS platforms.

Available applications are:

  • KJS - Offline King James with Strong Bible Search
  • KJV - Offline King James Bible Search
  • KJA - Offline King James Apocrypha Search
